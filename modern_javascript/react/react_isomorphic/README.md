isomorphic 同構，目的為server side rendering
解決兩個大問題

    1. SEO
    2. 前後端均為JS，可以做single-page applications（複數個單一頁面應用程式）

---

1.安裝套件

    npm install

2.初始化

    npm clean
    npm init

3.執行測試環境

    npm run dev

4.監控client端的動態改變

    npm watch:client

5.監控server端的動態改變

    npm watch:server


* Demo frontend: http://localhost:3000

* Demo backend: http://localhost:3000/api/example/test

(目前後端只有寫一個簡單的API）
